Put your exported .glb files here.

From Blender: File → Export → glTF 2.0 → format = glTF Binary (.glb)

Keep file sizes reasonable for the web — if a model is very dense,
use Blender's Decimate modifier or bake a lower-poly version for the
viewer. A few MB per model is fine; tens of MB will load slowly for
visitors on mobile connections.

After uploading a .glb here, add a matching entry in the admin panel
so it shows up on the site (see ../README.md).
