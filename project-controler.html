How this works (read once)
  1. Export your 3D piece from Blender as .glb (File → Export → glTF 2.0, format = glTF Binary).
  2. Upload that .glb file into the models/ folder of your website, using your host's file manager or FTP.
  3. Below, add a project entry and point Model file path at it, e.g. models/my-car.glb.
  4. Click Export projects.json — this downloads the updated data file.
  5. Upload that downloaded file into your website's data/ folder, replacing the old projects.json. That's the step that makes it visible to visitors — this panel edits a local draft only, it can't publish by itself on a plain file host.

Working on a new device? Use Import projects.json below and pick the current file from your site first, so you're editing the real list and not starting over.
